How to Permanently Fix Your App

To permanently fix the refreshToken issue in your application:

  1. Add this script tag to the <head> section of your index.html file (before any other scripts):

    <script src="/early-auth-fix.js"></script>

  2. Or store the fix in sessionStorage so it gets applied automatically:

    sessionStorage.setItem('indexHtmlFixScript', `
const script = document.createElement('script');
script.src = '/early-auth-fix.js';
document.head.appendChild(script);
`);

  3. For a long-term fix, add a refreshToken method to your AuthService implementation in the source code.
